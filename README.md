# Loan Default & Overview Dashboard

## 📊 Tools Used
- Power BI
- Excel

## 📌 Project Overview
This project presents an interactive Power BI dashboard analyzing loan data, focusing on applicant demographics, financial profiles, and risk metrics.

The goal is to identify patterns in loan distribution, borrower behavior, and default risk.

## 📂 Dataset
- Contains loan application data including income, employment type, loan purpose, and default status.

## 📈 Key Features of Dashboard
- Loan Amount by Purpose
- Average Income by Employment Type
- Default Rate (%) by Employment Type
- Average Loan Amount by Age Group
- Default Rate Trends Over Years

## 🔍 Key Insights
- Higher default rates are observed among certain employment groups.
- Full-time employees have higher average income compared to others.
- Loan amounts vary significantly by purpose (Business, Education, etc.).
- Default rates show fluctuations over time, indicating changing risk patterns.
